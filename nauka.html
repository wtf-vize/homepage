Znaczniki (h2)

Różnica pomiędzy znacznikami b a strong (h3)

Różnica pomiędzy znacznikami b i strong jest taka, że b(old) jedynie pogrubia tekst, który widzisz w przeglądarce. Znacznik strong natomiast nadaje tej treści większe znaczenie. Podkreśla istotność.

Znacznik a (h3)

Chyba najważniejszy znacznik w całym internecie. Bez niego nie było by SEO, nie było byp rzechodzenia pomiędzy stronami.

Skróty klawiaturowe w Visual Studio Code i nie tylko (h3)

  • Shift+Alt+F - formatowanie kodu
  • Shift+Alt+Strzałka w dół - powielenie aktualnej linijki
  • Ctrl+Shift+P - wyszukiwarka komend
  • Wykrzyknik (!) + Tab - Szkielet strony
  • code . - wpisane w wierszu poleceń uruchamia VSC
  • code index.html - wpisane w wierszu poleceń uruchamia VSC i od razu wskazany plik
